What's HTML

What is HTML?

HTML, or HyperText Markup Language, is the standard language used to create the structure of web pages. It uses tags and elements to organize text, images, links, tables, forms, and media so that web browsers can display them properly. HTML serves as the basic foundation of almost every website on the internet.

Breif History

HTML was first developed by Tim Berners-Lee in 1991 as part of the creation of the World Wide Web, allowing documents to be linked and shared through browsers. Over time, HTML evolved through several versions, including XHTML, until HTML5 became the modern standard in 2014 with improved multimedia and semantic features

HTML has developed greatly since its invention by Tim Berners-Lee in 1991. What began as a simple language for sharing linked scientific documents gradually expanded through multiple versions, adding tables, forms, images, stricter XHTML rules, and finally modern multimedia and semantic features in HTML5.
